### Markdown

Markdown 是一种轻量级且易于使用的语法，用于设置写作样式。它包括

```markdown
语法突出显示的代码块

- Bulleted
- List

1. Numbered
2. List

**粗体** 
_斜体_ 
`代码文本` 

[Link](url)链接写法 
![Image](src)图片插入
```

有关更多详细信息，请参阅[GitHub Flavored Markdown](https://guides.github.com/features/mastering-markdown/).
